This project works when a person moves near the sensor it detects the motion and automatically turns on, this supports electricity by consuming it less.

SCIGENIX School science fair
    Features that support the environment in this project:

    -it automatically turns off when there is no motion.

    -it only consumes 0.48 watts of power.

    -it can help to save electricity.
